    Run Local Agent

    Your camera is registered. Since it's on a local network, you need to run agent.py on any machine that can reach it. This agent pulls the RTSP stream and pushes it securely to the server.

    1 Install FFmpeg if not already installed
    2 Run agent.py from the repo

    ⚡ The stream will appear in the dashboard automatically once the agent connects. Keep agent.py running while the camera is in use.
